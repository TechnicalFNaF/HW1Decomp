# Five Nights at Freddy's: Help Wanted Decompilation

This is a full code decompilation of FNAF Help Wanted 1 (Vive1.0.11) that aims for identical functionality. This repository does not contain any game assets as you must obtain them legally via Steam or VivePort.

As this project aims for accuracy, it does not contain fixes or changes for blatant issues or bugs.

> [!WARNING]
> This project is currently incomplete and some gameplay aspects may not be fully functional.

## Prerequisites
- Unreal Engine 4.23.1 either from Epic Games Launcher or GitHub
- Visual Studio 2017 (Untested on other platforms)
